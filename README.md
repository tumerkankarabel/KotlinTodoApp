# KotlinTodoApp
A to do app written in kotlin that can perform operations such as adding, listing, searching, deleting and updating todo.

## Functions:
    - Listing Todos
    - Searching Todos
    - Todos Record
    - Deleting Todos
    - Update Todos

## Technologies
  - Dagger Hilt (Dependecy Injection)
  - MVVM (Model-View-ViewModel)
  - Navigation

## Database
  - Sqlite
  - Room
